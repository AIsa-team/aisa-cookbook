# 🧑‍🍳 The AISA.one Python Cookbook

Welcome to the collection of standalone "Recipes" for building agents with AISA.one.

**Philosophy:**
Each folder in this repository is a completely independent project. You can copy any folder out of this repo, install its specific requirements, and run it immediately.

## 🔑 Prerequisites
1. Get an API Key from [AISA.one](https://marketplace.aisa.one/?aff=GzQxs).
2. Set it in your environment:
   ```bash
   export AISA_API_KEY="sk-..."
   # Or create a .env file inside the recipe folder
   ```
